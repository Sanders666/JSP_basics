## JSP Study Repository

이 저장소는 **JSP 학습을 위한 예제와 노트들을 정리한 프로젝트**입니다.  
각 챕터별로 Servlet, Action Bean, Cookie, EL, JSTL,  
DB 연동, MyBatis, MVC 패턴 등을 **단계적으로 학습**할 수 있도록 구성되어 있습니다.

---

## Project Structure

- **chapter01_begin** : JSP 기초
- **chapter02_jsp** : JSP 기본 문법
- **chapter03_servlet** : Servlet 활용
- **chapter04_action_bean** : Action Bean
- **chapter05_cookie** : Cookie 처리
- **chapter06_EL** : Expression Language (EL)
- **chapter07_JSTL** : JSTL 활용
- **chapter08_DB ~ chapter10_DBCP** : 데이터베이스 연동
- **chapter11_mybatis** : MyBatis 사용
- **chapter12_FileUpDown** : 파일 업로드 / 다운로드
- **chapter13 ~ chapter17_MVC** : MVC 패턴 단계별 학습
- **chapter18_member** : 회원 관리 예제

---

## Learning Goals

- JSP와 Servlet의 기본 개념 이해
- 데이터베이스 연동 및 MyBatis 활용
- MVC 패턴을 통한 웹 애플리케이션 구조 학습
